# SKILL.md - 股票助手 (Stock Assistant)

## 触发词
- **"股票助手"** - 唤醒本项目
- **"分析A股"** - 分析A股股票
- **"分析美股"** - 分析美股股票
- **"帮我看看XX"** - 分析指定股票

## 功能
- **A股分析**：股票数据、技术指标、资金流向（无期权）
- **美股分析**：股票数据、期权策略、技术分析
- **智能路由**：自动识别 A股（6位数字）和美股（字母代码）

## 市场识别规则
| 代码格式 | 市场 | 分析内容 |
|----------|------|----------|
| 6位数字，以6开头 | A股上海 | 股票+技术面+资金流 |
| 6位数字，以0/3开头 | A股深圳/创业板 | 股票+技术面+资金流 |
| 大写字母1-5位 | 美股 | 股票+期权策略 |
| 6位数字ETF代码 | A股ETF | 股票+技术面+资金流 |

## 使用方式

### 命令行
```bash
cd /root/.openclaw/workspace/quant/StockAssistant
python main.py <代码>      # 分析股票
python main.py 000001      # 分析A股
python main.py TSLA        # 分析美股
python main.py 510050      # 分析A股ETF
```

### 在对话中触发
当用户说：
- "股票助手" → 启动本项目
- "分析一下腾讯" → 调用 A股分析（需要代码）
- "TSLA 期权策略" → 调用美股完整分析
- "帮我看看苹果" → 调用美股分析（需要代码）
