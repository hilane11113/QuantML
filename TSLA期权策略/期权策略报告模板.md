## 附录: 评分计算公式

```
综合评分 = RR因子(30%) + 流动性(20%) + 安全距离(20%) + 参数匹配(10%) + 舆情(20%)

其中:
- RR因子 = min(RR/2, 1) × 30
- 流动性 = min(OI×0.7+Vol×0.3, 20)
- 安全距离 = min(价差宽度%, 20)
- 舆情: 积极=20, 中性=10, 负面=0

VIX信号:
- 偏离度 = (VIX - MA10) / MA10 × 100%
- 绿灯: 偏离度 > 5%
- 黄灯: -5% <= 偏离度 <= 5%
- 红灯: 偏离度 < -5%
```
